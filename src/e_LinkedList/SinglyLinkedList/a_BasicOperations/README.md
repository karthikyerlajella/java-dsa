# Singly Linked List - Basic Operations

This section contains the fundamental operations performed on a Singly Linked List (SLL).

## Topics Covered

### Insertion

* Insert at Head
* Insert at Tail
* Insert at Position

### Deletion

* Delete Head Node
* Delete Tail Node
* Delete Node at Position

### Traversal

* Print Linked List
* Iterate Through Nodes

### Searching

* Search Element in Linked List

### Utility Operations

* Find Length of Linked List
* Check Empty List

---

## Time Complexity

| Operation      | Complexity |
| -------------- | ---------- |
| Insert at Head | O(1)       |
| Insert at Tail | O(n)       |
| Delete Head    | O(1)       |
| Delete Tail    | O(n)       |
| Search Element | O(n)       |
| Find Length    | O(n)       |
| Traverse List  | O(n)       |

---

## Learning Objective

Understand how nodes are connected using references and learn the core operations that form the foundation for advanced Linked List problems.

These concepts are frequently used in interview questions involving:

* Fast & Slow Pointers
* Reversing Linked Lists
* Cycle Detection
* Merging Lists
* Linked List Manipulation
